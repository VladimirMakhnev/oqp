# QUESTIONS.md — batched questions for the human (out-of-band)

> Write here ONLY truly blocking items (`CLAUDE.md §9`): broken build that cannot be repaired, or
> an irreducible physical/numerical ambiguity. For everything else, proceed on the best-justified
> assumption and document it in `PROGRESS.md` + the LaTeX audit doc. Do **not** interrupt the work
> stream per-question — batch them here and keep working on unblocked tasks.

## Format (copy this block per question)

```
### Q<n> — <one-line title>            [status: OPEN | ANSWERED]
- Date:        YYYY-MM-DD
- Phase/Task:  e.g. P2.3
- Blocking?:   yes/no — what is blocked
- Context:     what was tried, what the ambiguity is (cite paper/eq or file:line)
- Options:     A) … B) … (with the trade-off of each)
- Current assumption (used to keep moving): …
- Answer:      (human fills in)
```
